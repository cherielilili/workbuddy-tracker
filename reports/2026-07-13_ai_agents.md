# AI Agent 赛道日报 2026-07-13

## Manus

- 发布 Auto-Publish 功能（Build Once, Ship Continuously），支持一次构建后持续自动发布到多个平台。
- 来源：https://manus.im/blog

## OpenClaw

- 发布 v2026.7.1-beta.6 版本，亮点包括：
  - 新增模型和提供商：Featherless、Claude Sonnet 5、Mythos 5、Meta Muse Spark 1.1、ClawRouter；GPT-5.6 成为新安装默认模型。
  - 控制 UI 和原生 macOS 聊天：会话为主界面，带可搜索侧边栏、紧凑上下文环、推理努力滑块、原生 macOS 会话浏览器。
  - 对话式引导（Crestodian）：在 CLI、Web 安装和 macOS 应用中运行真实 Agent 循环设置。
  - 离线与语音移动聊天：iOS/Android 预缓存会话，Apple Watch 支持语音，iOS 支持 TTS 回复。
  - 会话组织与生成标题：通过工具模型生成简洁标题，支持分组、未读、重命名、分叉、归档、删除。
  - Telegram 和 Codex 连续性改进。
  - 启动与升级恢复改进。
- 来源：https://github.com/openclaw/openclaw/releases

## QwenPaw

- 发布 v2.0.0.post1 修复版本，主要修复包括：
  - 修复浏览器自动填充问题、暗色模式文本对比度、会话滚动加载停止、多层级孤立 tool_result 消息防御等。
  - 新增 /api/healthz 健康检查端点、可信代理验证、Windows 安全会话 ID 编码等。
- 来源：https://github.com/agentscope-ai/QwenPaw/releases
