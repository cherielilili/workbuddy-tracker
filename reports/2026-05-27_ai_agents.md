# AI Agent 赛道日报 2026-05-27

## 摘要
今日赛道多线并进：腾讯 WorkBuddy 连升两个次版本至 v4.24.1，引入自定义专家管理与 MCP 工具调用进度展示；阿里 QwenPaw v1.1.9 正式版发布，带来 Tauri 原生桌面应用与 Coding Mode；OpenClaw v2026.5.26 稳定版落地，涵盖 Gateway 优化与安全加固。

## 重大信号
- **[Tencent]** WorkBuddy v4.24.0 新增自定义专家管理能力与 MCP 工具调用进度展示，v4.24.1 进一步优化专家团协作调度
- **[Alibaba]** QwenPaw v1.1.9 正式版：Tauri 2.x 原生桌面应用（macOS/Windows）、三栏 Coding Mode Web IDE、Skill Market 跨供应商浏览安装
- **[OpenClaw]** v2026.5.26 稳定版：Gateway 启动优化、多渠道反应审批、Realtime Talk Web UI 控制、SSRF 安全加固

## 腾讯 WorkBuddy

- **v4.24.0**：新增自定义专家管理能力；支持 MCP 工具调用进度展示；新增连接器详情页与工作空间折叠功能；优化 Windows 新建任务速度 ([Changelog](https://www.codebuddy.cn/docs/workbuddy/Changelog))
- **v4.24.1**：优化共享任务列表与自动化任务删除流程；改进专家团协作任务调度、降低并发冲突；修复企业微信及 macOS 语言环境异常 ([Changelog](https://www.codebuddy.cn/docs/workbuddy/Changelog))

## 阿里 QwenPaw

- **v1.1.9 正式版**（2026-05-27）：发布 Tauri 2.x 原生桌面应用，支持 macOS 与 Windows；推出 Coding Mode，含文件树 + 标签页编辑器 + 内联 diff 审查三栏 Web IDE；上线 Skill Market，支持跨供应商浏览与安装；新增后台任务超时配置；统一渠道访问控制白/黑名单体系 ([Releases](https://github.com/agentscope-ai/QwenPaw/releases))

## OpenClaw

- **v2026.5.26 稳定版**（2026-05-27）：Gateway 启动优化，避免重复扫描插件/渠道/会话等服务；Transcript 集成支持会议摘要与来源分块；Signal、iMessage、WhatsApp 新增反应式审批；Discord 语音功能增强；Realtime Talk 可从 Web UI 与 Discord 检查、引导、取消；安全加固：浏览器快照遵守 SSRF 策略、系统事件无法伪造提示标记 ([Releases](https://github.com/openclaw/openclaw/releases))
