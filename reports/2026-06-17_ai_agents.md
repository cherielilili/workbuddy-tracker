# AI Agent 赛道日报 2026-06-17

## 摘要
今日赛道两项可见更新：阿里 QwenPaw 正式发布 v1.1.12 稳定版，完成控制台 UI 全面重构；行业分析平台 53AI 发布 OpenClaw 6.8 深度解读文章，聚焦 Agent 会话中断后任务恢复能力改进。

## 重大信号

- [Alibaba/QwenPaw] **QwenPaw v1.1.12 稳定版发布**：控制台模型页按提供商聚合重构，新增简洁扁平导航、宽屏布局、逐轮 Token 用量监控、越南语支持，完成从 beta.1 到正式版的迭代。来源：https://github.com/agentscope-ai/QwenPaw/releases

## 阿里 QwenPaw

- **v1.1.12 稳定版**（2026-06-17）正式发布，主要更新：
  - 控制台模型页按提供商聚合重构
  - 新增简洁模式（扁平导航）与宽屏布局切换
  - 逐轮 Token 用量实时监控
  - 新增越南语界面支持
  - 代码块语法高亮
  - 非阻塞自适应节流刷新
  - 桌面端口重启持久化
  - CI/CD 发布验证门控
  - 同日另发布 v1.1.12-beta.2 预发布版
  - 来源：https://github.com/agentscope-ai/QwenPaw/releases

## 行业横评

- **53AI**（2026-06-17）发布《OpenClaw 6.8 震撼发布：不堆噱头，彻底治愈 Agent 的宕机失忆症》，分析 OpenClaw v2026.6.8 在 Agent 输出格式混乱与进程中断后任务恢复方面的优化。来源：https://www.53ai.com/news/Openclaw/
