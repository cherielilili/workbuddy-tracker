# AI Agent 赛道日报 2026-06-12

## 摘要
今日赛道有两项重要动态：OpenClaw 发布 v2026.6.6 稳定版，安全边界全面加固；Manus 推出 Canva Connector，将设计能力整合进 Agent 工作流。WorkBuddy、QwenPaw 无新版本发布。

## 重大信号
- **[Manus]** Canva Connector 发布（2026-06-12）：用户可通过 Manus 直接调用 Canva 设计能力，融入自动化工作流 https://manus.im/blog
- **[OpenClaw]** v2026.6.6 稳定版发布（2026-06-12）：安全边界全面加固（transcript/沙箱/MCP stdio），Telegram 路由缓存优化，iMessage 持久化恢复，浏览器 WebSocket 验证改进 https://github.com/openclaw/openclaw/releases

## Manus
- （2026-06-12）**Canva Connector 发布**：Manus 正式接入 Canva，用户可将设计环节纳入 Agent 工作流，打通创意生产自动化链路。[来源](https://manus.im/blog)
- （2026-06-11）**客户案例**：Heicoders Academy 凭借 Manus 工作流以不足 20 人规模服务逾 12,000 名学员，是 Agent 规模化运营的典型实践。[来源](https://manus.im/blog)

## OpenClaw
- （2026-06-12）**v2026.6.6 稳定版发布**（前次最新稳定版为 v2026.6.5）：
  - 安全加固：transcript 与沙箱操作安全边界大幅收紧
  - Telegram 投递优化：路由与缓存机制改进，降低消息丢失风险
  - iMessage 恢复：引入持久化标记，会话中断后可靠恢复
  - 浏览器与 MCP 连通性：新增 WebSocket 校验
  - [来源](https://github.com/openclaw/openclaw/releases)
