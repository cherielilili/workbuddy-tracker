# AI Agent 赛道日报 2026-06-04

## 摘要
今日赛道整体平稳，阿里 QwenPaw 发布 v1.1.11-beta.1 测试版，新增小米 MiMo 内置渠道及飞书群会话共享模式；OpenClaw 推出 v2026.6.2-beta.1，优化插件安装策略与多平台频道稳定性。其余厂商无明显新动态。

## 阿里 QwenPaw

- **v1.1.11-beta.1 发布**：新增小米 MiMo 为内置模型渠道；飞书群会话新增共享会话模式；Telegram 工具审批支持交互式键盘；新增插件卸载钩子（Plugin Uninstall Hooks）；文件写入改用 aiofiles 非阻塞模式，提升 Windows 兼容性。（来源：https://github.com/agentscope-ai/QwenPaw/releases）

## OpenClaw

- **v2026.6.2-beta.1 发布**：插件安装机制从危险代码扫描改为运营商安装策略管理；Telegram、飞书、Discord、WhatsApp 多平台频道稳定性优化；流式文本保留与 ACK 时序修复；拒绝受损 Shell 快照与可疑配置以增强安全性。（来源：https://github.com/openclaw/openclaw/releases）
