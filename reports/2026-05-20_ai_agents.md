# AI Agent 赛道日报 2026-05-20

## 摘要
今日赛道有三项值得关注的更新：腾讯 WorkBuddy 推出 4.22.15 版本，进一步强化 MCP 连接器能力与产物微信分享功能；OpenClaw 正式发布 v2026.5.19 稳定版，新增自动代码审查、Meme 生成器及 Python 调试三项技能；阿里 QwenPaw 发布 v1.1.8.post1 补丁，修复 MCP 路由与 Windows 兼容性问题。

## 重大信号

- [Tencent] WorkBuddy 4.22.15 发布：MCP 连接器新增请求头配置与 User-Agent 支持，HTML 产物微信分享正式上线
- [OpenClaw] v2026.5.19 正式稳定版发布：autoreview、meme-maker、Python 调试三项技能同步落地，Gateway 启动延迟显著降低

## 腾讯 WorkBuddy

- **v4.22.15**（新版本）来源：[Changelog](https://www.codebuddy.cn/docs/workbuddy/Changelog)
  - MCP 连接器新增请求头配置能力，支持自定义 User-Agent
  - MCP 会话安装信任引导优化
  - Expert 可见性支持按网络环境配置
  - HTML 产物支持分享至微信
  - 对话执行进度展示优化
  - 上下文压缩及 Claw 功能多项稳定性修复

## 阿里 QwenPaw

- **v1.1.8.post1**（2026-05-20 补丁版）来源：[Releases](https://github.com/agentscope-ai/QwenPaw/releases)
  - 新增 OpenCode Go provider 集成（通过 meta.base_url_options）
  - 新增后台任务超时支持
  - MCP client 路由修复（路径含 '/' 的场景）
  - 计划面板暗色模式可见性修复
  - QwenPaw Pet Windows 兼容性与双语 UI 改善

## OpenClaw

- **v2026.5.19 稳定版**（2026-05-20 发布）来源：[Releases](https://github.com/openclaw/openclaw/releases)
  - 新增 'autoreview' 技能：代码收尾自动审查
  - 新增 meme-maker 技能：模板搜索与渲染
  - 新增 Python 调试技能：支持 pdb 及远程调试
  - HTTPS 代理支持，可配置 CA 信任
  - Gateway 启动延迟优化（边车初始化改进）
  - 内存搜索扫描改进，防止线程阻塞
  - QA-Lab 扩展：新增运行时一致性场景与个人 Agent 基准测试包
  - 最低 Node.js 22 版本要求提升至 22.19
