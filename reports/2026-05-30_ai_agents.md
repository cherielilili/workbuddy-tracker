# AI Agent 赛道日报 2026-05-30

## 摘要
今日赛道动态集中在三处：OpenClaw 2026.5.28 从 beta 周期毕业发布正式稳定版，带来运行时全面强化与多模型扩展；腾讯 WorkBuddy 接连推出 v4.24.2 与 v4.24.3，新增微信分享、OAuth 应用管理等功能；Manus 发布跨国儿童合规场景客户案例。

## 重大信号

- **[OpenClaw]** OpenClaw 2026.5.28 正式稳定版发布（2026-05-30）：从 beta 周期毕业，agent 及 Codex 运行时状态分离与清理机制全面增强，新增 Claude Opus 4.8、NVIDIA 模型目录与 GitHub Copilot agent 运行时集成，40+ 贡献者参与本轮发布。来源：https://github.com/openclaw/openclaw/releases

## 腾讯 WorkBuddy

- **v4.24.3**（新版）：优化会话列表，隐藏专家团子成员与展开箭头，列表聚焦度提升；修复技能安装失败、自定义模型请求参数异常、工具栏与浮层层叠问题；同步界面权限与进程实际权限；优化 Windows 菜单栏与侧边栏样式。[来源](https://www.codebuddy.cn/docs/workbuddy/Changelog)
- **v4.24.2**（新版）：新增微信分享、项目邀请与 OAuth 应用管理；侧边栏折叠按钮与任务置顶/归档/筛选功能；支持 MITM 代理 HTTPS 信任；加入工具调用循环检测机制；WorkBuddy 微信 ClawBot 消息支持 Markdown 渲染。[来源](https://www.codebuddy.cn/docs/workbuddy/Changelog)
