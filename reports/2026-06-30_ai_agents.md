# AI Agent 赛道日报 2026-06-30

## 摘要
OpenClaw 发布 v2026.6.11 稳定版，新增 Slack 中继模式、Mattermost 命令支持及插件外置分发；53AI 报道 OpenClaw 与 Cursor 同日发布移动端原生应用，标志 AI Agent 正式进入移动端日常使用场景。WorkBuddy 持续修复，最新 v5.1.7 修复多项 UI 与兼容性问题。

- [OpenClaw] OpenClaw v2026.6.11 稳定版发布，新增 Slack 中继模式、Mattermost 命令等

## 腾讯 WorkBuddy
- 发布 v5.1.7（06-26），修复设计创意模式画布刷新按钮无反应、Mac 微信小程序白屏、小猫头像误触、技能卡片渲染崩溃、Windows 中文用户名安装后无法打开等多项问题

## OpenClaw
- 发布 v2026.6.11 稳定版（06-30），亮点包括：Slack 中继模式、原生 Mattermost /oc_queue 命令、每 DM 模型覆盖、openclaw agent --message-file 文件驱动、RAFT CLI wake bridge、插件外置分发、Android 设置详情面板、Codex 部分 delta 修复等
- 来源：https://github.com/openclaw/openclaw/releases
- 53AI 报道：OpenClaw 与 Cursor 同日发布移动端原生应用，OpenClaw 推出适配 iOS、安卓的全功能原生 App，采用本地优先架构，支持用户在移动端远程操控私有 AI Agent
- 来源：https://www.53ai.com/news/Openclaw/
- 53AI 发布文章《运维界的 OpenClaw 来了！》，介绍运维领域开源 AI 原生智能体 Ongrid，可自动完成运维问题排查
- 来源：https://www.53ai.com/news/Openclaw/
