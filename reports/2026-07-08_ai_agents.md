# AI Agent 赛道日报 2026-07-08

## 摘要

QwenPaw 发布 v2.0.0-beta.4，增强审批卡片 UI 与定时任务安全开关；OpenClaw 发布 v2026.7.1-beta.2，新增 GPT-5.6 支持、Telegram Codex 工作流及 iOS 原生应用更新；Stanford HAI 被纽约时报报道 AI 改造纽约州政府老旧系统。

## 阿里 QwenPaw

- QwenPaw 发布 v2.0.0-beta.4（07-08），主要更新：增强审批卡片 UI 和 UX；为定时任务添加工具安全开关以控制审批；修复滚动保护、长文本上传配置、MCP 运行时审批级别等问题。

## 开源 OpenClaw

- OpenClaw 发布 v2026.7.1-beta.2（07-05），亮点包括：支持 GPT-5.6 模型家族；新增 `openclaw attach` 命令用于外部 harness 附加；Telegram 支持 Codex 配对与运行控制；新增事件驱动 cron（on-exit 调度）；iOS 原生应用采用 iOS 26 视觉系统，改进聊天、通话、引导流程；iMessage 原生投票功能；Mac 本地 Gateway 自动安装；Control UI 侧边栏导航、推理强度滑块等。

## Stanford HAI

- 纽约时报报道（07-08）：AI 正在改造纽约州政府老旧系统，如传真和电报要求，HAI 被提及。

## 行业动态

- 53AI 发布文章《运维界的 OpenClaw 来了！》（06-30），介绍开源运维 AI 智能体 Ongrid，支持自然语言排查、自动编排工具、集群态势展示，可一行命令自托管。
- 53AI 发布文章《刚刚，OpenClaw 和 Cursor 杀入手机！Agent 从此塞进口袋》（06-30），报道 OpenClaw 与 Cursor 同日发布移动端原生应用。
