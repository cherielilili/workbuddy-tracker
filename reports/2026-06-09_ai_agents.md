# AI Agent 赛道日报 2026-06-09

## 摘要
今日赛道动态集中于 OpenClaw 稳定版正式落地、WorkBuddy 功能增补与 Manus 生态延伸三条主线。OpenClaw 结束本轮 beta 周期发布 v2026.6.5 stable，引入 SQLite 状态持久化与并行网络搜索等重要架构升级；WorkBuddy v5.0.3 补强腾讯文档个人版集成；Manus 推出 Zoom Connector，将会议内容纳入工作流自动化覆盖范围。

## 重大信号
- [OpenClaw] **v2026.6.5 stable 正式发布**（2026-06-09）：结束 beta 周期，核心变更包括 SQLite 状态持久化、并行 web 搜索集成、Anthropic extended-thinking 异常恢复、QQBot 推理内容过滤及 Matrix 渠道语音线程支持。来源：https://github.com/openclaw/openclaw/releases

## OpenClaw
- **v2026.6.5 stable**（2026-06-09）正式发布，前次快照为 beta.2，主要变更：
  - QQBot 渠道在消息投递前自动过滤模型推理内容
  - MCP 工具返回结果强制类型转换，提升兼容性
  - Anthropic extended-thinking 模式异常恢复机制
  - 并行 web 搜索集成
  - Google Vertex ADC 鉴权修复
  - Matrix 渠道新增语音与线程支持
  - 认证与插件状态迁移至 SQLite，提升持久性与可靠性
  - agent/tool/provider 三层校验逻辑收严
  - macOS Node 运行时稳定性改善
  - 版本升级路径安全加固
  - 来源：https://github.com/openclaw/openclaw/releases

## 腾讯 WorkBuddy
- **v5.0.3**（2026-06-08）发布，较 v5.0.2 新增：
  - 腾讯文档个人版导入能力上线，支持内容自动刷新
  - 新增'成长计划'用户入口
  - Expert 代理页面性能优化
  - 多项 UI 与稳定性问题修复
  - 来源：https://www.codebuddy.cn/docs/workbuddy/Changelog

## Manus
- **Zoom Connector 发布**（2026-06-09）：新增 Zoom 集成，可将用户有权访问的 Zoom 会议内容（决策记录、客户反馈、跟进事项等）自动转化为工作流触发源，延伸 Manus 在会议场景的任务自动化覆盖能力。来源：https://manus.im/blog

## 阿里 QwenPaw
- **v1.1.11-beta.2** 发布，较 beta.1 新增：
  - 浏览器控制升级，支持页面坐标点击操作
  - CDP 超时参数配置与浏览器配置文件隔离
  - MCP 服务工具白名单及 UI 开关
  - 控制台插件扩展基础架构
  - 零配置免费模型接入与 OAuth 认证支持
  - 来源：https://github.com/agentscope-ai/QwenPaw/releases
