# AI Agent 赛道日报 2026-07-09

## 摘要

Manus 发布 Branch 功能，支持单上下文多并行方向；QwenPaw 发布 v2.0.0-beta.5，新增内置搜索工具和 Azure Bot 频道插件；Stanford HAI 发布 Biomni AI 实验室伙伴及 AI 加速科学发现两篇报道。

## 重大信号

- [Manus] Introducing Branch: One Context, Many Parallel Directions — 新产品功能发布，允许用户在单一上下文中创建多个并行分支，提升多方向探索效率。

## Manus

- 发布 Branch 功能（07-09）：支持在单一上下文中创建多个并行方向，用户可同时探索不同路径，提升多任务处理效率。
  - 来源：https://manus.im/blog

## QwenPaw

- 发布 v2.0.0-beta.5（07-09）：新增内置 web_search 和 web_fetch 工具；新增 Azure Bot 频道插件，支持 i18n/自定义图标/文档 URL；修复多项滚动、浏览器检测、审批处理等问题；优化推理内容对齐和上下文阈值显示。
  - 来源：https://github.com/agentscope-ai/QwenPaw/releases
