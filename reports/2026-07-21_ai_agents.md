# AI Agent 赛道日报 2026-07-21

## 摘要

今日赛道动态集中在产品迭代：QwenPaw 发布 v2.0.1-beta.1，新增 PawApp SDK 和可编辑 Agent 模式；OpenClaw 发布 v2026.7.2-beta.3，引入远程编码会话和移动端自动化；WorkBuddy 社区出现多篇用户实操文章，但官方 Changelog 无更新；Stanford HAI 发布法律 AI 透明度问题研究。

## 重大信号

- [QwenPaw] QwenPaw v2.0.1-beta.1 发布，新增 PawApp SDK 和可编辑 Agent 模式
- [OpenClaw] OpenClaw v2026.7.2-beta.3 发布，引入远程编码会话和移动端自动化
- [Stanford HAI] 法律 AI 透明度问题研究发布

## 阿里 QwenPaw

- 发布 v2.0.1-beta.1（07-21），主要更新包括：
  - 新增 PawApp SDK 和看板应用
  - 支持用户可编辑 Agent 模式
  - 增强 ReMe 记忆索引稳定性
  - 修复 MCP 迁移、本地模型 GGUF 检查、子 Agent 审批路由等问题
  - 来源：https://github.com/agentscope-ai/QwenPaw/releases/tag/v2.0.1-beta.1

## 开源 OpenClaw

- 发布 v2026.7.2-beta.3（07-18），主要更新包括：
  - 远程编码会话：支持在云 Worker 上运行 Control UI 会话，在终端中打开 Codex 和 Claude 目录会话
  - 移动端自动化：将 Automations 功能带到移动端，Android 新增前台语音唤醒
  - 引导式设置：从 Settings 配置模型提供商，通过引导页面设置渠道
  - 新增 Linux deb 和 AppImage 安装包
  - 来源：https://github.com/openclaw/openclaw/releases/tag/v2026.7.2-beta.3

## Stanford HAI

- 发布新研究《法律 AI 的透明度问题》（07-20），指出语言模型在法律任务中的错误风险，呼吁从机构视角实现有用且稳健的透明度
  - 来源：https://hai.stanford.edu/news/legal-ais-legibility-problem

## 腾讯 WorkBuddy

- 社区出现多篇用户分享文章，包括续费报告攻略、使用感受等，但官方 Changelog 无新版本发布
  - 来源：https://cloud.tencent.com/developer/search/article?q=WorkBuddy
