# AI Agent 赛道日报 2026-06-01

## 摘要
今日赛道动态集中于阿里 QwenPaw 与开源项目 OpenClaw。QwenPaw v1.1.10 正式发布稳定版，新增腾讯元宝渠道集成与飞书 Thread 回复能力；OpenClaw 发布 v2026.6.1-beta.1，强化 Skill Workshop 控制界面并将 GitHub Copilot 外置为官方插件，同步在 53AI 收录两篇深度解析文章。

## 重大信号
- [Alibaba/QwenPaw] QwenPaw v1.1.10 稳定版发布，新增腾讯元宝渠道集成与飞书 Thread 回复
- [OpenClaw] v2026.6.1-beta.1 发布，Skill Workshop 获完整 Control UI，GitHub Copilot 外置为官方插件

## 阿里 QwenPaw
- **v1.1.10 稳定版**（2026-06-01）正式发布：新增 spawn_subagent 工具支持临时工作区执行、Open Directory 本地项目引用、腾讯元宝渠道集成、飞书 Thread 回复、动态上传限额，及 E2E 测试迁移与集成测试扩展。[发布页](https://github.com/agentscope-ai/QwenPaw/releases)
- **v1.1.10-beta.2**（2026-06-01）：网站头部样式优化、更新时技能保留机制、聊天输入增强、腾讯元宝 Bot 集成。

## OpenClaw
- **v2026.6.1-beta.1**（2026-06-01 09:45 UTC）：Agent 与 CLI 运行时恢复增强，涵盖中断工具调用、过期会话绑定、压缩切换及媒体投递重试；Skill Workshop 获完整 Control UI（提案管理与修订处理）；Tokenjuice 与 GitHub Copilot 外置为官方插件；新增 iOS 推送中继、iPad 布局及 Workboard 编排工具。[发布页](https://github.com/openclaw/openclaw/releases)
- **v2026.5.31-beta.4**（2026-06-01 02:04 UTC）：Gateway 新增 Tailscale Serve 服务名绑定、Control UI 暴露通信通知设置、插件安装索引持久化至 SQLite、MiniMax M3 模型支持，iMessage 监控状态迁移至 SQLite。
