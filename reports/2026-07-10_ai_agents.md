# AI Agent 赛道日报 2026-07-10

## 摘要
今日赛道动态集中在产品版本更新：阿里 QwenPaw 发布 v2.0.0 正式版，引入 Runtime 2.0 核心架构、Scroll Context 上下文管理、终端 TUI 等重大特性；OpenClaw 发布 v2026.7.1-beta.2，新增 GPT-5.6 支持、Telegram Codex 工作流、事件驱动 cron 等；腾讯 WorkBuddy 持续小版本迭代，修复多项稳定性问题。Stanford HAI 发布 AI 加速科学发现专题报道。

## 重大信号
- [QwenPaw] QwenPaw v2.0.0 正式发布：核心架构升级为 Runtime 2.0，引入 Scroll Context 上下文管理、终端 TUI、长期记忆 ReMe v0.4.0 等重大特性

## 阿里 QwenPaw
- **v2.0.0 正式版发布**（07-10）：核心架构升级为 Runtime 2.0，基于 AgentScope 2.0 重构内核；新增 Agent OS Driver 统一管理 MCP/A2A/ACP 连接；引入 Loop Engineering 三种模式（Normal/Mission/Goal）；新增 Governance & Sandbox 策略引擎。上下文管理方面，推出 Scroll Context（SQLite 后端 + FTS5 全文搜索）、Native Compression、长期记忆 ReMe v0.4.0 后端。新增 Terminal UI（TUI）全屏终端聊天应用，支持流式输出、工具调用渲染、会话管理。Console 和 UI 方面，新增审批卡片、长文本上传、会话级工具审批、移动端响应式布局等。模型提供商新增 OpenAI Response API 支持，DashScope 更新内置模型列表。
  - 来源：https://github.com/agentscope-ai/QwenPaw/releases/tag/v2.0.0

## OpenClaw
- **v2026.7.1-beta.2 发布**（07-05）：新增 OpenAI GPT-5.6 模型支持；新增外部 harness 附加功能（openclaw attach）；Telegram 支持 Codex 配对工作流（/login 启动、steer 运行、恢复回复）；新增事件驱动 cron（on-exit schedule 类型）；iOS 原生应用采用 iOS 26 视觉系统，优化 Chat/Talk/onboarding/reconnect 流程；iMessage 新增原生投票创建/读取/投票功能；新增能力配置文件实现会话级工具和访问边界；macOS 应用可自动安装本地 Gateway；Control UI 新增会话优先侧边栏、紧凑上下文计量器、暖色主题、推理努力滑块等。
  - 来源：https://github.com/openclaw/openclaw/releases/tag/v2026.7.1-beta.2

## 腾讯 WorkBuddy
- **v5.2.3 版本发布**（07-06）：新增模型夜间折扣展示；优化腾讯文档授权流程；优化输入框工具栏折叠与发送后高度恢复；修复弱网断连后对话无法恢复、Claw 多端消息错发、MCP 会话丢失、连接器切换误报失败、项目任务切换后工具调用丢失、消息流式输出重复拼接、Windows 产物右键打开失败等问题。
  - 来源：https://www.codebuddy.cn/docs/workbuddy/Changelog
- **v5.2.2 版本发布**（07-03）：新增用量提醒与到期提醒；优化启动关键路径；修复系统休眠唤醒后通信通道不可用、会话恢复时后端重启竞态、暂停后历史消息丢失、MCP 授权参数兼容性、企业 Skill 更新后本地内容仍为旧版等问题。
  - 来源：https://www.codebuddy.cn/docs/workbuddy/Changelog
- **v5.2.1 版本发布**（07-02）：优化启动链路；优化专家导航入口样式；技能广场支持搜索企业自建 Skill；修复首次安装白屏、切换任务/会话异常导致白屏、腾讯文档上传进度不准确、大文件预览导致客户端卡死等问题。
