# AI Agent 赛道日报 2026-05-25

## 摘要
今日赛道增量平稳，主要来自 QwenPaw 与 OpenClaw 的 beta 迭代。QwenPaw 发布 v1.1.9-beta.1，引入 coding mode 与 Tauri 桌面端支持；OpenClaw 在稳定版 v2026.5.22 基础上推进第二轮 beta，v2026.5.24-beta.2 于昨晚深夜发布。

## 阿里 QwenPaw
- **v1.1.9-beta.1**（2026-05-25）发布：新增 coding mode 功能；引入 Tauri 2.x 桌面应用打包支持；修复 Gemini API token 映射异常；优化微信消息处理逻辑；控制台插件支持安装/卸载并自动重载页面。([release 页](https://github.com/agentscope-ai/QwenPaw/releases))

## OpenClaw
- **v2026.5.24-beta.2**（2026-05-24 23:49）发布：在 beta.1 基础上进一步扩展 QA-Lab 测试覆盖范围；持续优化 Gateway 缓存与启动性能；更新 Signal、Telegram 等多通道集成文档。([release 页](https://github.com/openclaw/openclaw/releases))
