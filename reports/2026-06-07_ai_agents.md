# AI Agent 赛道日报 2026-06-07

## 摘要
今日赛道动态集中于 OpenClaw：开源版 v2026.6.5-beta.2 于当日发布，带来 Anthropic 扩展思考会话恢复、MCP 工具结果类型转换修复及并行 Web 搜索集成等增量改进。其余主流平台（WorkBuddy、Manus、Coze Studio、QwenPaw）今日无新动态。

## OpenClaw

- **v2026.6.5-beta.2**（2026-06-07）来源：https://github.com/openclaw/openclaw/releases
  - QQBot 在消息下发前自动剥离模型推理内容，避免推理标签暴露给终端用户
  - MCP 工具结果强制类型转换，修复向 Anthropic 接口发送非法格式时触发的报错
  - Anthropic 扩展思考模式（extended-thinking）下，缓存过期后会话可自动恢复，不再中断
  - 并行 Web 搜索集成，支持 API Key 自动发现
  - 修复 Google Vertex ADC 模型解析异常
