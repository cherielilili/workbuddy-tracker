# AI Agent 赛道日报 2026-06-16

## 摘要
今日赛道有两处增量更新：阿里 QwenPaw 发布 v1.1.12-beta.1，带来安全隔离、多语言及模型管理页面重构等多项改进；OpenClaw 推出 v2026.6.8 稳定版，正式落地 Telegram/WhatsApp 富文本表格及 GLM-5.2、Claude Haiku 4.5 支持。其余各厂商今日无新动态。

## 阿里 QwenPaw

- **v1.1.12-beta.1**（2026-06-16 预发布）[发布页](https://github.com/agentscope-ai/QwenPaw/releases)
  - 安全：每次安装独立隔离 Keychain 主密钥，防止跨实例泄露
  - 控制台：记忆搜索工具样式重构，新增越南语支持，宽屏模式切换
  - 对话：逐轮 token 与上下文用量浮层，空响应兜底消息
  - 模型页：全页重构，按 Provider 聚合，统一卡片 UI
  - 技能市场：集成更新，UI 优化与斜杠命令注入改进
  - 渠道：腾讯元宝新增引用消息与机器人过滤，DingTalk 存活看门狗
  - 性能：全渠道非阻塞 flush + 自适应节流

## OpenClaw

- **v2026.6.8 稳定版**（2026-06-16）[发布页](https://github.com/openclaw/openclaw/releases)
  - Telegram 与 WhatsApp 正式支持结构化富文本（表格、列表、可折叠引用块）
  - Agent 与 Gateway 容错恢复机制增强
  - 新增模型支持：GLM-5.2、Claude Haiku 4.5
  - Usage Hooks：原生完整页脚渲染器、默认模板、定点数格式化
  - UI 及移动端稳定性修复
- **v2026.6.8-beta.2**（2026-06-16 预发布）：功能与稳定版同步，含 WhatsApp ACP 绑定支持
