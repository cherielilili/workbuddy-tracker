# AI Agent 赛道日报 2026-05-15

## 摘要
今日赛道多点更新：WorkBuddy 发布 4.22.12 版本，新增腾讯会议与企业微信连接器卡片；OpenClaw 连续推出两个 beta 版本，强化 WhatsApp、语音及 Telegram 能力；53AI 同日刊发 OpenClaw 分层治理下篇，并报道 AgentScope Java 1.1.0 Harness Framework 正式发布。

## 重大信号
- **[Tencent / WorkBuddy]** v4.22.12 新增腾讯会议与企业微信连接器卡片，并上线设置面板网络诊断
