## OpenClaw

- 发布 v2026.7.1-beta.2（2026-07-05）：
  - 新增 OpenAI GPT-5.6 模型支持
  - 新增 `openclaw attach` 命令，支持外部 harness 附加到已有 Gateway 会话
  - Telegram 支持 Codex 工作流配对、运行控制和回复恢复
  - 新增事件驱动 cron（on-exit 调度），支持命令退出时唤醒 agent
  - iOS 原生应用更新，采用 iOS 26 视觉系统，优化聊天、通话、引导和重连流程
  - iMessage 新增原生投票创建、读取和投票功能
  - 新增能力配置文件，支持按会话设置工具和访问边界
  - macOS 应用可自动安装和启动本地 Gateway
  - 控制 UI 新增会话优先侧边栏、紧凑上下文计量器、暖色主题、推理难度滑块等
  - 新增 ClawRouter 提供商插件，支持凭据作用域动态模型发现
  - 模型覆盖：新增 GPT-5.6、Nemotron Super 1M 上下文窗口支持
