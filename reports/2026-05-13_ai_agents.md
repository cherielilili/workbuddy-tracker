# AI Agent 赛道日报 2026-05-13

## 摘要
今日赛道整体节奏平稳，阿里 QwenPaw 与开源项目 OpenClaw 均在当日推出新 beta 版本。QwenPaw 新增 OAuth 2.1 PKCE 远程 MCP 鉴权支持，OpenClaw 则集中强化安全合规与媒体管控能力。其余厂商今日无新动态。

## 阿里 QwenPaw
- **v1.1.7-beta.2**（2026-05-13）：新增通过插件注册 FastAPI APIRouter 实例；支持 OAuth 2.1 PKCE 用于远程 MCP 服务器鉴权；BaseChannel 新增流式输出 hooks 并适配企业微信（WeCom）。来源：https://github.com/agentscope-ai/QwenPaw/releases

## OpenClaw
- **v2026.5.12-beta.5**（2026-05-13）：网关协议改进，支持聊天帧流式传输；修复 GitHub Copilot 图像理解 OAuth Token 交换问题；设备配对新增审批要求（涵盖 setup-code 与浏览器访问）；WhatsApp/飞书/Line 媒体下载强制文件大小限制；Telegram 轮询卡顿检测与 Token 轮换；CLI 与网关均新增会话敏感信息脱敏能力。来源：https://github.com/openclaw/openclaw/releases
