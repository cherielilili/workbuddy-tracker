# AI Agent 赛道日报 2026-05-18

## 摘要
今日赛道动态集中于自动化能力升级与版本迭代：Manus 正式发布 Scheduled Tasks 2.0，将自动化触发机制扩展至时间驱动与事件驱动两类模式；OpenClaw 推出 v2026.5.18 稳定版，新增三项技能扩展；QwenPaw 发布 v1.1.8-beta.1，强化 Matrix E2EE 与企业微信集成。

## 重大信号
- [Manus] Scheduled Tasks 2.0 发布：支持定时任务与事件驱动自动化，覆盖重复性工作及 Manus 构建 Web App 的触发场景（https://manus.im/blog）
- [OpenClaw] v2026.5.18 稳定版发布：新增 autoreview 自动审查、Meme 制作、Python pdb 调试三项技能，浏览器增加模态对话框处理（https://github.com/openclaw/openclaw/releases）

## Manus
- **Scheduled Tasks 2.0**（2026-05-18）：发布定时任务 2.0，支持时间触发的周期性工作自动化及基于事件的 Manus 构建 Web App 触发机制，显著扩展 Agent 自动化调度能力。[查看原文](https://manus.im/blog)

## 阿里 QwenPaw
- **v1.1.8-beta.1**（2026-05-18）：升级 agentscope 依赖至 1.0.20；新增 Matrix E2EE 验证与企业微信审批处理增强；支持按模型聚合 Token 用量统计；修复 SSE 连接泄漏及浏览器工具生命周期管理问题。[查看 Release](https://github.com/agentscope-ai/QwenPaw/releases)

## OpenClaw
- **v2026.5.18 稳定版**（2026-05-18）：正式稳定发布，新增 autoreview 自动审查、Meme 制作（支持模板搜索与渲染）、Python pdb 远程调试三项技能；浏览器新增待处理模态对话框支持及 blockedByDialog 返回值；Mac 设置页面重构为统一卡片布局；ProxyLine 升级至 0.3.3。[查看 Release](https://github.com/openclaw/openclaw/releases)
- **v2026.5.18-beta.1**（2026-05-18）：与稳定版同日发布的测试版，内容基本一致。
