# AI Agent 赛道日报 2026-07-05

## 摘要
OpenClaw 发布 v2026.7.1-beta.2，新增 GPT-5.6 支持、Telegram Codex 工作流及事件驱动 cron 等多项特性；QwenPaw 发布 v2.0.0-beta.2，引入滚动上下文管理器与 cron 更新命令；WorkBuddy 持续修复大量问题，版本迭代至 5.1.7；53AI 报道 OpenClaw 与 Cursor 移动端原生应用上线。

## 重大信号
- [OpenClaw] OpenClaw 发布 v2026.7.1-beta.2，新增 GPT-5.6 支持、Telegram Codex 工作流、事件驱动 cron 等，标志 Agent 框架持续向多模态、事件驱动演进。
- [QwenPaw] QwenPaw 发布 v2.0.0-beta.2，引入滚动上下文管理器，提升长对话记忆能力。
- [Industry] 53AI 报道 OpenClaw 与 Cursor 同日发布移动端原生应用，AI Agent 正式进入移动端日常使用场景。

## 腾讯 WorkBuddy
- 发布 v5.1.7（2026-06-26），修复多项 UI 和兼容性问题，包括设计创意模式画布刷新、Mac 微信小程序白屏、Windows 中文用户名安装等。
- 此前 v5.1.6 新增财报数据内置技能和 Token Plan 模型选项；v5.1.5 优化产物面板展示和目录权限检查。
- 近期版本持续修复大量问题，涵盖沙箱安全、连接器授权、Teams 协作、自动化任务等。

## 字节 Coze
- Coze Studio 开源版 v0.5.1 于 2026-02-05 发布，新增 NATS EventBus 实现、会话隔离、知识库 OpenAPI 等，修复 SQL 注入等安全问题。
- 无新版本发布。

## 阿里 QwenPaw
- 发布 v2.0.0-beta.2（2026-07-02），引入滚动上下文管理器（持久化历史+召回 REPL）、cron 更新命令、DashScope 模型列表更新至 Qwen3.7 和 DeepSeek V4 等。
- 修复多项问题，包括内存系统提示词优化、工具审批通知推送、桌面截图等。

## Manus
- 博客最新文章为 2026-06-30 的《Mobbin connector on Manus》，介绍 Mobbin 连接器上线。
- 此前 06-24 发布 Hosting Modes 服务器配置匹配功能。
- 无新动态。

## OpenClaw
- 发布 v2026.7.1-beta.2（2026-07-05），亮点包括：
  - OpenAI GPT-5.6 支持
  - 外部 harness 附加（openclaw attach）
  - Telegram Codex 工作流（/login 配对、steer 运行、恢复回复）
  - 事件驱动 cron（on-exit 调度）
  - iOS 原生应用适配 iOS 26 视觉系统
  - iMessage 原生投票
  - 安全作用域对话（capability profiles）
  - macOS 本地 Gateway 自动安装
  - Control UI 导航改进
- 53AI 报道《刚刚，OpenClaw和Cursor杀入手机！Agent从此塞进口袋》，介绍 OpenClaw 推出移动端原生应用。
