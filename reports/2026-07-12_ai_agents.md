# AI Agent 赛道日报 2026-07-12

## 摘要

WorkBuddy 持续高频迭代，本周发布 v5.2.3 修复多项稳定性问题；Manus 推出 Branch 功能，支持并行上下文分支；OpenClaw 发布 v2026.7.1-beta.5，新增对话式引导、ClawRouter 路由与配额、原生 macOS 聊天 UI 及 GPT-5.6 支持；QwenPaw 发布 v2.0.0 正式版，核心架构升级为 Runtime 2.0；Stanford HAI 发布 AI 加速科学发现专题报道及 Biomni AI 实验室伙伴报道。

## 重大信号

- [Manus] Introducing Branch: One Context, Many Parallel Directions — 支持在单一上下文中创建多个并行分支，提升多任务处理灵活性。
- [OpenClaw] OpenClaw 2026.7.1-beta.5 发布 — 新增对话式引导、ClawRouter 路由与配额、原生 macOS 聊天 UI、GPT-5.6 支持等重大特性。
- [QwenPaw] QwenPaw v2.0.0 正式发布 — 核心架构升级为 Runtime 2.0，引入 Scroll Context、TUI、ReMe v0.4.0 等重大特性。

## 腾讯 WorkBuddy

- 发布 v5.2.3（07-06）：新增模型夜间折扣展示，优化腾讯文档授权流程，修复弱网断连后对话恢复、Claw 多端消息错发、MCP 会话丢失等多项问题。
- 社区新增多篇用户实操文章：续费报告攻略、自动化拼多多日报、WorkBuddy 使用心得等。

## Manus

- 发布 Branch 功能（07-09）：支持在单一上下文中创建多个并行分支，用户可同时探索不同方向，提升多任务处理效率。

## 字节 Coze

- Coze Studio 开源版无新版本发布，最新仍为 v0.5.1（2025-02-05）。

## 阿里 QwenPaw

- 发布 v2.0.0 正式版（07-10）：核心架构升级为 Runtime 2.0，引入 Scroll Context（SQLite 历史记录 + FTS5 全文搜索）、TUI（终端聊天界面）、ReMe v0.4.0 长期记忆后端、Loop Engineering 新循环门设计、治理与沙箱策略引擎等。

## OpenClaw

- 发布 v2026.7.1-beta.5（07-11）：新增对话式引导（Crestodian AI 引导设置）、ClawRouter 路由与配额管理、原生 macOS 聊天 UI（会话侧边栏、推理滑块等）、GPT-5.6 支持（Sol/Terra/Luna 模型族）、外部 harness 附加、Telegram Codex 工作流等。

## Stanford HAI

- 发布专题报道《How AI Is Accelerating Scientific Discovery》（07-08）：探讨 AI 工具在假设生成、实验设计、模式发现方面的作用。
- 发布报道《Stanford Scientists Build an AI Lab Partner》（07-09）：介绍 Biomni AI 实验室伙伴，可分析医疗数据、设计实验。
- 活动预告：7/15 NVIDIA RAPIDS 数据科学工作坊、7/23 Rising Stars in Data Science 信息会。
