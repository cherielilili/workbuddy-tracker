# AI Agent 赛道日报 2026-06-03

## 摘要
今日赛道最重要动态为 OpenClaw 发布 v2026.6.1 正式稳定版，引入 Skill Workshop 提案审核与 MiniMax M3 模型支持；WorkBuddy 发布 v4.24.8 修复上下文压缩系列问题；Manus 新增一篇 HubSpot+Gmail 销售流程整合客户案例。

## 重大信号
- **[OpenClaw]** v2026.6.1 正式稳定版发布（2026-06-03），新增 Skill Workshop 提案审核、MiniMax M3 模型、多平台 Channel 稳定性全面提升、SQLite 状态管理，Tokenjuice 与 GitHub Copilot 插件完成外置化。[release 链接](https://github.com/openclaw/openclaw/releases)

## OpenClaw（开源）
- **v2026.6.1 稳定版**（2026-06-03 19:35 UTC）正式发布，经 beta.1/beta.2/beta.3 三轮迭代后转正。核心新增：Skill Workshop 提案审核界面、Tokenjuice 与 GitHub Copilot 插件外置化、MiniMax M3 模型接入、SQLite 后端支撑 iMessage 监控与插件索引、iPad 布局适配；Telegram/WhatsApp/iMessage/Slack/Discord/Teams/Google Chat 多平台 Channel 稳定性全面增强，热路径冗余操作优化。[release 链接](https://github.com/openclaw/openclaw/releases)
- **v2026.6.1-beta.3**（2026-06-03 09:16 UTC）同日发布，为正式版最终预发布验证构建，功能集与稳定版一致。
