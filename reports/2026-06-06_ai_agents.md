# AI Agent 赛道日报 2026-06-06

## 摘要
今日赛道更新集中于 OpenClaw，发布 v2026.6.5-beta.1，新增并行网页搜索集成、Matrix 语音/Thread 支持及 SQLite 持久化认证状态等特性。其余跟踪源（WorkBuddy、Manus、Coze Studio、QwenPaw）均无新增动态。

## OpenClaw

- **v2026.6.5-beta.1**（2026-06-06）[GitHub Release](https://github.com/openclaw/openclaw/releases)
  - QQBot 频道剥离模型推理标签（reasoning tags），避免原始思维链内容外泄至消息正文
  - MCP 工具返回值强制类型转换（result coercion），提升跨工具兼容性
  - Anthropic extended-thinking 异常恢复机制上线，减少长思考链中断
  - 并行网页搜索（Parallel web-search）集成，提升信息检索效率
  - Google Vertex ADC 认证修复
  - Matrix 频道新增语音消息与 Thread 支持
  - 认证状态与插件状态存储切换至 SQLite 持久化方案，提升重启耐久性
