# AI Agent 赛道日报 2026-07-07

## 摘要

今日赛道动态集中在产品版本更新：腾讯 WorkBuddy 发布 v5.2.3，优化模型夜间折扣展示与腾讯文档授权流程；阿里 QwenPaw 发布 v2.0.0-beta.3，引入 Windows 原生沙箱、速率限制增强等多项特性；OpenClaw 发布 v2026.7.1-beta.2，新增 GPT-5.6 支持、Telegram Codex 工作流、事件驱动 cron 等。Stanford HAI 公布 7 月活动安排，包括 NVIDIA RAPIDS 数据科学工作坊等。

## 重大信号

- [QwenPaw] QwenPaw v2.0.0-beta.3 发布：引入 Windows 原生沙箱、速率限制增强、记忆后端可禁用等重大更新。
- [OpenClaw] OpenClaw v2026.7.1-beta.2 发布：新增 GPT-5.6 支持、Telegram Codex 工作流、事件驱动 cron、iOS 原生应用更新等。

## 腾讯 WorkBuddy

- **v5.2.3 发布（2026-07-06）**：新增模型夜间折扣展示，费用信息更透明；优化腾讯文档授权流程，减少登录和授权失败；优化输入框工具栏折叠与发送后高度恢复，小窗口输入体验更稳定；优化云助理工具卡头像、会话展示与创建助理时的仓库/分支选择体验；修复弱网断连后下一轮对话无法恢复、Claw 多端对话消息错发、MCP 会话丢失后无法恢复、连接器快速切换或重连时误报连接失败、项目任务切换后工具调用展示丢失、自定义专家/专家团成员映射异常、消息流式输出重复拼接、Windows 产物右键打开文件失败等多项问题。
  - 来源：https://www.codebuddy.cn/docs/workbuddy/Changelog

## 阿里 QwenPaw

- **v2.0.0-beta.3 发布（2026-07-07）**：引入 Windows 原生沙箱实现；增强速率限制，提供多维保护；新增 'none' 内存后端以禁用记忆系统；重构技能相关 UI；修复移动端聊天历史面板空会话列表、流式完成时间显示、MCP 客户端配置容错、GitHub Models 新端点支持、ACP 版本锁定兼容性、cron 作业时区记录、OpenRouter OAuth 路由恢复等问题；新增请求超时、重试和 AbortSignal 支持；支持插件注册自定义频道及 schema 驱动配置 UI；添加使用感知的自动搜索和后端特定嵌入。
  - 来源：https://github.com/agentscope-ai/QwenPaw/releases

## OpenClaw

- **v2026.7.1-beta.2 发布（2026-07-05）**：新增 OpenAI GPT-5.6 模型支持；新增外部 harness 附件命令 openclaw attach；新增 Telegram Codex 工作流，支持 /login 配对、控制 Codex 运行及恢复回复；新增事件驱动 cron 运行（on-exit 调度）；iOS 原生应用采用 iOS 26 视觉系统，更新 Chat、Talk、onboarding 和重连流程；iMessage 新增原生投票创建、阅读和投票功能；新增能力配置文件，实现按会话的工具和访问边界；macOS 应用可自动安装和启动本地 Gateway；控制 UI 新增会话优先侧边栏、紧凑上下文计量器、暖色主题、推理努力滑块、简化 composer 和斜杠命令选择器；新增 ClawRouter 提供商插件，支持凭据作用域动态模型发现、OpenAI 兼容及原生 Anthropic/Gemini 传输、预算报告；新增 Nemotron Super 1M 上下文窗口支持；新增 OpenRouter 认证头保留；新增节点上下文路径支持、设备批准恢复指南、软恢复 CLI 会话等。
  - 来源：https://github.com/openclaw/openclaw/releases
