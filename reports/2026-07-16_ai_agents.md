# AI Agent 赛道日报 2026-07-16

## 摘要
今日赛道动态集中在产品迭代：WorkBuddy 发布 v5.2.3 版本，优化模型夜间折扣展示与腾讯文档授权流程；QwenPaw 发布 v2.0.0.post2，新增聊天归档、sudo 权限请求及 ReMe 记忆系统可靠性提升；OpenClaw 发布 v2026.7.2-beta.1，引入远程编码会话、原生自动化移动端及 Linux 安装包。Stanford HAI 发布 AI 主权悖论报告及 AI 心理健康安全测试缺陷研究。

## 重大信号
- [Tencent] WorkBuddy v5.2.3 发布：新增模型夜间折扣展示，优化腾讯文档授权流程
- [Alibaba] QwenPaw v2.0.0.post2 发布：新增聊天归档、sudo 权限请求、ReMe 记忆系统可靠性提升
- [Open Source] OpenClaw v2026.7.2-beta.1 发布：新增远程编码会话、原生自动化移动端、Linux 安装包
- [Stanford HAI] AI 主权悖论报告：评估国家购买、构建或租赁 AI 以维持战略控制
- [Stanford HAI] AI 心理健康安全测试缺陷研究：专家对 AI 安全响应评估一致性低

## 腾讯 WorkBuddy
- WorkBuddy 发布 v5.2.3 版本（2026-07-06）：新增模型夜间折扣展示，费用信息更透明；优化腾讯文档授权流程，减少登录和授权失败；优化输入框工具栏折叠与发送后高度恢复；修复弱网断连后对话恢复、连接器误报等问题。
- 腾讯云开发者社区新增多篇用户实操文章：包括续费报告攻略、WorkBuddy 配置连接 ima 笔记、拼多多多店铺日报自动化等。

## 阿里 QwenPaw
- QwenPaw 发布 v2.0.0.post2（2026-07-14）：新增聊天归档功能；新增 sudo 权限请求；修复沙箱 PATH 保留；提升 ReMe 记忆系统可靠性、可观测性及 CJK 嵌入安全性；新增基于 token 的 Windows 沙箱；优化工具守卫规则。

## 开源 OpenClaw
- OpenClaw 发布 v2026.7.2-beta.1（2026-07-15）：新增远程编码会话（在云 worker 上运行 Control UI 会话）；原生自动化移动端（Android 语音唤醒、相机/位置/通知能力）；引导式 Control UI 设置；Linux deb 和 AppImage 安装包；MCP 隔离；Discord 语音频道参与者变化通知；Skill Workshop 扫描历史会话生成技能建议。

## Stanford HAI
- 发布《AI 主权悖论》报告（07-14）：评估国家购买、构建或租赁 AI 以维持战略控制，调查商业主权解决方案减少对美国科技巨头依赖的程度。
- 发布 AI 心理健康安全测试缺陷研究（07-13）：发现 AI 开发者依赖人类专家评估 AI 心理健康响应安全性，但专家对安全定义一致性低。
- 发布 Biomni AI 实验室伙伴报道（07-09）：Biomni 可分析海量医疗数据、发现模式、设计实验，加速疾病研究。
