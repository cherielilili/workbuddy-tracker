## 阿里 QwenPaw
- **v2.0.0-beta.1（2026-06-26）** 重大架构升级：
  - 底层从 AgentScope 1.x 全面迁移至 2.0，采用模块化架构与增强工具调用协调机制
  - 引入 Agent OS Driver，提供外部能力统一抽象接口
  - 记忆系统重构，切换至 ReMe4（基于轮次跟踪）
  - 新增 Slack 频道支持、OpenAI Response API 提供商集成
  - 改进 MCP 工具与 GLM 模型兼容性；控制台多页面移动端适配
  - Terminal chat UI 内置至 qwenpaw（Phase 1）
  - 警告：早期 Beta，含破坏性变更，不建议用于生产环境
  - 来源：https://github.com/agentscope-ai/QwenPaw/releases
