# AI Agent 赛道日报 2026-06-19

## 摘要

今日赛道动态集中于 WorkBuddy 和 OpenClaw 两条主线：腾讯 WorkBuddy 密集推送 v5.1.0–v5.1.4，引入微信支付嵌入式插件和 Teams 文件资产库等重要功能；OpenClaw 发布 v2026.6.9-beta.1，集成 Codex 并将官方 Provider 插件拆分为独立 npm 包，375 个 PR 合并入库。阿里 QwenPaw 与 Manus 各有小幅增量更新。

## 重大信号

- **[Tencent]** WorkBuddy v5.1.1 上线微信支付嵌入式插件，支持 AI 驱动的支付发起，并新增 Token Plan 模型配置项。
- **[Tencent]** WorkBuddy v5.1.0 正式推出 Teams 项目协作文件资产库，支持 deeplink 页面跳转，可从外部直接启动任务/项目。
- **[OpenClaw]** v2026.6.9-beta.1（06-19）引入 Codex 深度集成、Official Provider 插件独立 npm 发布，单周期合并 375 个 PR，Telegram 富文本交付全面升级。

## 腾讯 WorkBuddy

新版本密集发布（v5.1.0–v5.1.4），覆盖功能扩展与稳定性修复：

- **v5.1.0（06-13）**：新增对话消息多选删除；支持 deeplink 页面跳转（可从外部直接启动任务/项目）；个人与项目云存储扩容；Teams 项目协作引入文件资产库。（来源：https://www.codebuddy.cn/docs/workbuddy/Changelog）
- **v5.1.1（06-14）**：上线微信支付嵌入式插件，支持 AI 驱动支付发起；新增 Token Plan 模型配置项；对话中加入模型切换视觉分隔线；助手对话自动注入当前时间。（来源：https://www.codebuddy.cn/docs/workbuddy/Changelog）
- **v5.1.2（06-17）**：修复历史任务恢复时权限确认弹窗反复出现、专家召唤退出后无法重新选择、专家团队任务异常中断后会话续接等问题。（来源：https://www.codebuddy.cn/docs/workbuddy/Changelog）
- **v5.1.3（06-17）**：修复 Windows 中文用户名环境下自我修复程序兼容性问题。（来源：https://www.codebuddy.cn/docs/workbuddy/Changelog）
- **v5.1.4（06-18）**：修复对话加载动画持续残留和左侧面板转圈、粘贴内容展示截断及表格遮挡、历史任务打开空白页报错等问题。（来源：https://www.codebuddy.cn/docs/workbuddy/Changelog）

## OpenClaw

- **v2026.6.9-beta.1（06-19）**：Telegram 交付层全面升级（HTML 渲染 + Markdown 保留）；Agent 恢复机制增强（重试 + 历史会话修复）；Codex 深度集成（自动插件审批）；Official Provider 插件独立拆分为 npm 包发布；Web 与 Native 客户端 Control UI 优化；本周期合并 375 个 PR。（来源：https://github.com/openclaw/openclaw/releases）
